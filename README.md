# Menual

Snap, translate, and eat! Your best pocket menu manual.

Menual is a lightweight, frontend-only menu translator. Paste your Gemini API key, add menu photos, and translate menu items into Traditional Chinese or English.

## Features

- Static app built with only `index.html`, `style.css`, and `script.js`
- Gemini API key stored locally and optionally passed with `#key=...`
- Multiple image upload with thumbnails, per-image analysis, retry, and delete
- Flash / Lite Gemini model switch, with Lite fallback for busy Flash requests
- Optional image compression before upload
- Mobile-friendly result cards with original text, translation, price, notes, and image search
